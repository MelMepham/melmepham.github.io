JAVASCRIPT LANGUAGE

It's all Language!

Monday the 27st of November

So my understanding is JavaScript is all about syntax! what do you use where and why!?! so lets break some of these down a little…

Parentheses ( ) So, like math these can be used as a grouping operator to have more control over what bits of code is executed first. remember in Math hoe 4 * 3 - 5 = 7 however 4 * (3 - 5) = -8. This is because in the second equation the parentheses executes first.
The other reason for using these is with functions. so a typical function is written function myFunction() and I can then put parameters in the parentheses such as function myFunction (a, b) a and b are things that change how the function operates.

Next we have Brackets [ ] These are used mainly with arrays to store chunks of code you then use within your code. I have only really come into contact with brackets when assigning arrays i.e myArray [1, 2, 3, 4] and when getting information out of an array i.e that would get the third (JS stars counting at 0) out of the array in this case that would = 3.

Braces! {} These are fun, they make up the skeleton of your code, which makes it easier for you and others to read through and understand when things start and things end. They are used to store functions, i.e function myFunction (a, b) { a + b } This means when you call the function myFunction (2, 4) it will execute the braces and the result in this case will = 6.

“ “, ‘ ‘ double and single quote marks.. usually there is not much if a difference with these two, they are used to store strings i.e var myVar = “name” (single quotes marks would have been fine also) the only way they differ is when you need to put a string in a string i.e “my name is ‘Mel’” I haven’t come into this yet so I couldn’t explain it well enough as I could if I had used it. but even in this instance it doesn’t matter what part you put the singles or doubles around.
