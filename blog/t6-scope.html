SCOPE

Are we talking local or global?

Wednesday 20th December

So, there is a thing called scope.. it has many different meanings, but the one we are talking about today is one that refers to code and in particular JavaScript.

Scope is used in JavaScript to talk about variables, variables are super important in JavaScript, they give programs functions elements to work with, using a variable inside a function means that variable only has that value in that function, so if you call the variable outside that function it won’t work. If you set the variable outside a function it is placed in the global scope, which means you can use it in functions, the issue with this however is if you then try to define the variable within a function it can do weird things, and not always good weird.. so its best to stay away from using variables globally, and keep ‘em local!
